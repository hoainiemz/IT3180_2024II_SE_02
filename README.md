Để chạy được ứng dụng, thực hiện lần lượt các bước sau:

B2: Mở terminal lên.

B3: Chạy lệnh: "mvn clean install".

B4: Sau khi thực hiện 3 bước trên, bạn có thể chạy thử ứng dụng bằng lệnh: "mvn spring-boot:run".
